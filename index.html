We have to download git bash, begore this we have a github account

Now we can use git any where like command Line or In any code Editor (like VS Code) After this we'll do first thing is configure our git using this

git config --global user.name " your Name "

git config --global user.email "your mail@email.com"

If we want to check that our email and name is add or not than we can

git config --list
