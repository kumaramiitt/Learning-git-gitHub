Status

This is use to check the status of code

Here status has four types

  • Untracked
  • unmodified
  • modified
  • staged

Untracked : means we have created file or repo but we did not add it

Unmodified : its the normal form or stage of files. It will not show

Modified : when we change something or update in our code it is used to call Modified

Staged : its the middle stage of add and commit, it means when we add something then after this there a condition to stage the file
